*** Notes
- We separate:
  - ACTIVITY is a project with many tasks
    - TASK is a work item in an activity or within another task (in which case it is a SUBTASK)
  - ASSIGNATION is a time block with a clear beginning and end time that cannot be moved by auto-scheduling.
      It can be one of the following:
    - APPOINTMENT: a TASK with a fixed start and end that cannot be moved by auto-rescheduling
    - ACTIVITY BLOCK: a time slot dedicated to a specific ACTIVITY.
    - GENERAL BLOCK: a time slot dedicated to a set of activities (Work, Home, etc.)
  - TASK SLOT is a re-schedulable slot allocated to a specific task.  Not visible on the calendar directly.
    It must be contained in a suitable ACTIVITY BLOCK or GENERAL BLOCK (but might only be partially contained,
    if the task has greater expected effort than allowed by the length of the block).
